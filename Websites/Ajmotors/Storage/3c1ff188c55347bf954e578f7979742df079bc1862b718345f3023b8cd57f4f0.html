# 2014 Toyota Prius Alpha S touring selection
{Stock number. 23841}

The 2014 Toyota Prius Alpha S Touring Selection is a hybrid vehicle that was part of Toyota's Prius family lineup, and it specifically refers to a more upscale trim level of the Prius Alpha, which was available in markets like Japan and Europe.

# Key features
- Push start button: The push start button in the 2014 Toyota Prius Alpha S Touring Selection is part of the Keyless Entry and Start system, which allows you to start the vehicle without needing to insert a traditional key into the ignition.

- Multi function steering wheel: The multi-function steering wheel in the 2014 Toyota Prius Alpha S Touring Selection (and other modern Toyota models) is designed to allow the driver to control a variety of vehicle functions without taking their hands off the wheel.

- EV mode: The EV Mode in the 2014 Toyota Prius Alpha S Touring Selection is a feature that allows the car to operate solely on electric power for short distances, reducing fuel consumption and emissions when driving at low speeds.

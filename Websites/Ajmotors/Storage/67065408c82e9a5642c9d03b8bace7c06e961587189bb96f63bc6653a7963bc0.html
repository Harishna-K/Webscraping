# Key Features

- Cruise Control : Maintains a constant speed for comfortable highway driving.

- Reversing Camera : Provides a view of the area behind the vehicle to assist in reversing safely.

- Bluetooth : Enables hands-free phone calls and audio streaming.

- Auto Start Stop : Automatically shuts off the engine when the vehicle is stationary to conserve fuel, restarting when the accelerator is pressed.

- Memory Seats : Recalls and adjusts seating positions for multiple drivers with the push of a button.

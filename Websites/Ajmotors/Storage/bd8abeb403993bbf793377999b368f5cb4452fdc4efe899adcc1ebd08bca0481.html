# 2014 Subaru WRX S4 Eyesight AWD
{Stock number. 25597}

The 2014 Subaru WRX S4 Eyesight AWD is a performance sedan that combines Subaru's iconic rally-inspired WRX heritage with the convenience and safety features of modern technology. The "S4" designation is typically used in the Japanese market, and it refers to a more refined and feature-packed version of the WRX.

# Key features
- Eyesight: Subarus Eyesight is an advanced driver assistance system (ADAS) that uses a pair of stereo cameras mounted on the front windshield to monitor the road ahead. It's designed to enhance safety by providing features that help prevent accidents or reduce their severity.

- Pre crush safety: If the system detects an impending collision, it can automatically apply the brakes to help prevent or lessen the impact.


- LDW: If the car starts to drift out of its lane without signaling, the system alerts the driver.

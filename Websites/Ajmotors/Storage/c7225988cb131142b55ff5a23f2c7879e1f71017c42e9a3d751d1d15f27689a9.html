# 2012 Toyota Vellfire HYBRID 4WD
{Stock number. 23131}

The Vellfire Hybrid typically combines a 2.4-litre gasoline engine with an electric motor, offering a smooth and efficient driving experience.

# Key features
- Cruise control: Ideal for highway driving, reducing the need for constant speed adjustments.

- Captain seats: Unlike a traditional bench seat, captain's seats provide two separate seats, allowing for more personal space and comfort.

- Power sliding door: Ideal for families, allowing easy access to the rear seats without needing to open traditional swing doors.

This 2019 Toyota Corolla Sport G Hybrid hatchback in black, on16 inch alloy wheels, has travelled only 35,600 kilometres.
It has a 1.8 litre twin cam petrol/hybrid engine with CVT transmission, that provides good performance with 4.5-star 3.3L/100km fuel economy.
The Corolla has long been the benchmark for good reliable cars, and this later version carries on that tradition, with many upgrades.
It features 7 airbags, ABS, stability/traction control, forward collision mitigation, and lane departure warning, all contributing towards its 5-star safety rating.
Other features include CD, cruise control, smart key starting/locking, and a large information screen which includes a rear-view camera.

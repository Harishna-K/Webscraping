# Honda Vezel HV sensing
{Stock number. 22786}

The Honda Vezel HV Sensing is a hybrid version of Honda's compact SUV, known for its stylish design and efficient performance. The "HV" stands for hybrid vehicle, combining a gasoline engine with an electric motor to improve fuel efficiency and reduce emissions.

# Key features
- Honda sensing: Contains safety features like Pre crash safety, Cruise control, and Lane keeping warning.

- Electronic hand brake: This can be integrated with other safety features, such as automatic hold functions, which keep the vehicle stationary on inclines.

- Dual climate: control is a feature in many modern vehicles that allows the driver and front passenger to set different temperature settings for their respective sides of the cabin.

# 2013 Toyota Corolla Fielder Hybrid G
{Stock number. 24390}

The 2013 Toyota Corolla Fielder Hybrid G is a compact hybrid station wagon, part of the Toyota Corolla Fielder lineup in Japan. The "G" designation generally represents a higher trim level, offering more features and a higher level of equipment compared to the base models.

# Key features
- Eco mode: Eco Mode slows down throttle response, meaning the accelerator pedal will feel less responsive. This is done to discourage rapid acceleration, which can lead to higher fuel consumption.

- Push button start:
The Push Button Start system is a convenient and modern feature commonly found in many vehicles, including the 2013 Toyota Corolla Fielder Hybrid G. It allows you to start and stop the engine without needing to use a traditional key in the ignition.

- Multi function steering wheel: The multi-function steering wheel in the 2013 Toyota Corolla Fielder Hybrid G is a convenient feature that allows the driver to control various vehicle functions without taking their hands off the wheel.

# 2012 Toyota Aqua G
{Stock number. 24045}

The 2012 Toyota Aqua G (also known as the Toyota Prius C in some markets) is a hybrid subcompact car. It is part of Toyota's Prius family and is designed to be an affordable, fuel-efficient option for urban driving.

# Key features
- Push button start: The Push Button Start system, often referred to as Keyless Start, is a feature commonly found in modern vehicles, including the 2012 Toyota Aqua G (also known as the Toyota Prius C in some markets). This system allows the driver to start the engine without needing to insert a traditional key into the ignition.

- Multi function steering wheel: The Multi-Function Steering Wheel in the 2012 Toyota Aqua G (also known as the Toyota Prius C in some markets) is a feature designed to enhance convenience and safety by allowing the driver to control various vehicle functions without taking their hands off the steering wheel.

- ECO mode: The Eco Mode in the 2012 Toyota Aqua G (also known as the Toyota Prius C in some markets) is a driving mode designed to optimize fuel efficiency and reduce fuel consumption by adjusting certain vehicle settings. It is part of Toyota's effort to make their hybrid vehicles more environmentally friendly and economical to operate.

# 2010 Lexus HS 250h

{Stock number. 17278}
The 2010 Lexus HS 250h is a luxury hybrid sedan known for its fuel efficiency and comfortable ride. It features a 2.4-litre four-cylinder engine paired with an electric motor, delivering a combined output of around 187 horsepower.

# Key features
- Electric seats: power-adjustable front seats, which can include features like lumbar support and memory settings for added comfort and convenience.

- Cruise control: Allows you to set a desired speed, which the car will maintain until you deactivate it by braking or turning off the system.

- Key less entry: Allows you to unlock the doors and start the engine without removing the key fob from your pocket or bag.

If you're looking for more information about the car, feel free to reach out! Whether you have questions about specific features or need assistance with anything else, our team is happy to help.

# 2012 Honda CR-Z Hybrid
{Stock number. 24142}

(Car details intro)The 2012 Honda CR-Z Hybrid is a compact, sporty hybrid car that blends eco-friendly performance with a fun-to-drive spirit.

# Key features
- Push start button: The push start button in the 2012 Honda CR-Z (and many modern vehicles) is a feature that allows you to start the engine without needing to insert a traditional key into the ignition.

- Sport mode: Sport Mode in the 2012 Honda CR-Z transforms the car from a fuel-efficient hybrid to a more lively and engaging compact sports car. Its a great feature for those who want to have fun behind the wheel while still benefiting from the hybrid powertrains efficiency.

- Paddle gear shifter: The paddle gear shifter in the 2012 Honda CR-Z is a feature available on models equipped with the Continuously Variable Transmission (CVT). While the CR-Z is primarily designed as a hybrid vehicle with an automatic transmission, the paddle shifters give drivers a more interactive and engaging experience by allowing them to manually control gear-like shifts within the CVT.

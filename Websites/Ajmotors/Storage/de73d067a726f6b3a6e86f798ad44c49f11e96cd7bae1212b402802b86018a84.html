# 2017 Nissan Note E-Power Nismo
{Stock number. 24709}

The 2017 Nissan Note E-Power Nismo is a performance-oriented variant of the Nissan Note, featuring Nissan's innovative e-Power hybrid technology paired with the sporty Nismo trim. The e-Power system uses a gasoline engine to generate electricity to charge the battery, which then powers an electric motor that drives the wheels.

# Key features
- LDW: LDW is a valuable safety feature that helps reduce the chances of accidents caused by distraction or drowsiness while driving. However, it is not a substitute for attentive driving.

- Pre crush safety:
Pre-Crash Safety refers to a suite of advanced driver assistance systems (ADAS) that are designed to prevent or mitigate the effects of a collision before it occurs. These systems typically rely on a combination of sensors, cameras, radar, and sometimes lidar to detect potential hazards, like other vehicles or obstacles, and take action to avoid or reduce the severity of a crash.

- Push button start:
Push Button Start is a convenience feature found in many modern vehicles, including the 2017 Nissan Note E-Power Nismo. It allows the driver to start and stop the engine with the simple press of a button, rather than using a traditional key or turning a key in an ignition switch.
